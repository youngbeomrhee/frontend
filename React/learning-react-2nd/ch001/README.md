# 명령어 정리
## 초기화
```npm init -y```

### 초기화확인
```vi package.json```

## 설치
```npm i underscore```

### 설치확인
```
vi package.json
ll ./node_modules
```

### 의존성 확인
```
vi package-lock.json
ll ./node_modules
```

## 삭제
```npm remove react```

### 삭제 및 의존성 확인
```
vi package.json
vi package-lock.json
ll ./node_modules
```
